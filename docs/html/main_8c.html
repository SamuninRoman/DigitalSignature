Функция загрузки ключа из файла.

Загружает ключ из файла в формате PEM. Тип ключа (открытый или закрытый) определяется параметром type.

Аргументы
filenameИмя файла, из которого будет загружен ключ.
typeТип ключа: "pub" для открытого ключа, "priv" для закрытого ключа.
Возвращает
Указатель на структуру KeyData, содержащую загруженный ключ.

Функция сохранения ключа в файл.

Сохраняет открытый и закрытый ключи в отдельные файлы в формате PEM. Имена файлов формируются путем добавления "_pub.pem" и "_priv.pem" к переданному имени файла.

Аргументы
filenameИмя файла, к которому будут добавлены суффиксы "_pub.pem" и "_priv.pem" для формирования имен файлов.
keyУказатель на структуру KeyData, содержащую ключ.
